Рецепты

Справочник «Рецепты» используется для ведения списка рецептов упаковки и маркировки ГП, которые применяются при операциях сериализации и агрегации упаковок ГП. Находится в разделе «Справочники» > «Рецепты».

Описание справочника

Страница справочника включает следующие элементы:

Панель фильтров

Для справочника накладных предусмотрен поиск по номеру накладной, дате создания, объекту, контрагенту. Для этого необходимо:

1) ввести значение в поля «Номер», «Дата», «Объект», «Контрагент»;

2) нажать на кнопку «Поиск»;

3) для удаления заданных критериев поиска в фильтрах нажать кнопку «Сбросить».

Список записей

В справочнике предусмотрена возможность поиска номенклатуры по следующим параметрам или их комбинации:

1) «Код продукта» - выбирается из выпадающего списка кодов продукции, содержащихся в справочнике «Номенклатура» (возможно выбрать несколько значений кодов);

2) «Название» - полное или частичное;

3) «Упаковщик» - выбирается из выпадающего списка производителей (возможно выбрать несколько производителей);

4) «Собственник» - выбирается из выпадающего списка собственников (возможно выбрать несколько собственников);

5) «Владелец ТМ/Держатель РУ» - выбирается из выпадающего списка производителей;

6) «GTIN» - полный или частичный;

7) «Штрих-код» - полное числовое значение штрих-кода (13 символов).

Перед заведением номенклатуры убедитесь, что выполнили следующие действия:

  • Правильно выбрали товарную группу в левом верхнем углу.

  • Заполнены справочники «Контрагенты» и «Объекты», связанные с данной номенклатурой.

  • Создали нужный записи справочника «Рецепты».

1. Откройте окно создания новой записи

  1. Перейдите в раздел «Справочники» > «Номенклатура».

  2. В правом верхнем углу окна справочника нажмите кнопку «Создать».

2. Заполнение полей в формах «Создание номенклатуры»

Обратите внимание: набор вкладок в форме зависит от Товарной Группы (ТГ), которую вы выбрали.

Особенности и ограничения

  • После создания номенклатуры ее можно найти в общем списке справочника, используя панель фильтров.
  • Редактирование номенклатуры возможно только после завершения всех производственных заказов, в которых она указывалась. Для редактирования существующей записи нажмите на три точки в конце строки записи, в выпадающем списке нажмите кнопку «Редактировать».
  • Удаление номенклатуры возможно только при наличии прав и если для нее не создавался заказ кодов. Для удаления нажмите на три точки в конце строки записи, в выпадающем списке нажмите кнопку «Удалить».
