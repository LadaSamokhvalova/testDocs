#### Список записей
В справочнике предусмотрена возможность поиска номенклатуры по следующим параметрам или их комбинации:

1) «Код продукта» - выбирается из выпадающего списка кодов продукции, содержащихся
в справочнике «Номенклатура» (возможно выбрать несколько значений кодов);

2) «Название» - полное или частичное;

3) «Упаковщик» - выбирается из выпадающего списка производителей (возможно выбрать
несколько производителей);

4) «Собственник» - выбирается из выпадающего списка собственников (возможно выбрать
несколько собственников);

5) «Владелец ТМ/Держатель РУ» - выбирается из выпадающего списка производителей;

6) «GTIN» - полный или частичный;

7) «Штрих-код» - полное числовое значение штрих-кода (13 символов).
